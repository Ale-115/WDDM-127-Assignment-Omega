WDDM-127 Assignment 2 - Recipes site

This project is a web application that helps users discover random recipes and search for meals using TheMealDB API.

Features
- Display a random recipe with image, title, and instructions
- Search for recipes by keywords
- Show search results with images and names
- View recipe details (ingredients)

Structure
- `index.html` - Main HTML file
- `style.css` - CSS styles for layout, grids, and design
- `script.js` - JavaScript for random recipe and search functionalities
- `assets/` - Folder for images and resources

Usage
1. Open `index.html` in your browser
2. Click "Show Me a Recipe" to get a random meal
3. Use the search bar to find recipes by introducing a keyword.
